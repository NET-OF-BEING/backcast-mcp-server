# Check Plan Status

Help the user review their backcasting plan progress:

1. **List Plans**: Use `backcast_list_plans` to show all saved plans.

2. **Load Plan**: If the user specifies a plan, use `backcast_load_plan` to load it.

3. **Show Progress**: Use `backcast_get_overview` to display:
   - Overall completion percentage
   - Steps completed vs remaining
   - Blocked items

4. **Next Actions**: Use `backcast_get_next_actions` to show what the user should work on next.

5. **Analyze**: If requested, use `backcast_analyze_plan` for deeper insights and optimization suggestions.

## Quick Commands

- "What's my progress?" → Show overview and percentage
- "What should I do next?" → Show next actionable steps
- "Analyze my plan" → Full analysis with suggestions
